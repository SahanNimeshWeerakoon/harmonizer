# Music Theory Basics

## What Are Musical Notes?

Music is built from **notes**.  
A note is a sound with a specific pitch (frequency).

In Western music, the basic notes are:

- C
- D
- E
- F
- G
- A
- B

After B, the sequence repeats again at a higher pitch:

C D E F G A B C

This repeating pattern is called an **octave**.

---

# Semitones and Whole Tones

The distance between notes matters.

## Semitone
The smallest interval in Western music.

Example:

- C → C#
- E → F

## Whole Tone
Two semitones.

Example:

- C → D
- F → G

---

# Sharps and Flats

## Sharp (#)
Raises a note by one semitone.

Example:

- C# = one semitone above C

## Flat (b)
Lowers a note by one semitone.

Example:

- Db = one semitone below D

---

# Scales

A scale is a collection of notes arranged in order.

## Major Scale Formula

Whole - Whole - Half - Whole - Whole - Whole - Half

Example: C Major

C D E F G A B C

---

# Chords

A chord is multiple notes played together.

The most common chord is the **triad**.

## Major Chord Formula

- Root
- Major 3rd
- Perfect 5th

Example: C Major chord

- C
- E
- G

## Minor Chord Formula

- Root
- Minor 3rd
- Perfect 5th

Example: A Minor

- A
- C
- E

---

# Melody

A melody is the main musical line people sing or remember.

Example melody:

C - E - G - F - E

Melody is usually horizontal (notes over time).

---

# Harmony

Harmony is additional notes played together with the melody to support it.

Melody = main voice  
Harmony = supporting voices

If melody is:

C

Possible harmonies could be:

- E
- G

Together they create a C major chord:

C + E + G

---

# Intervals

An interval is the distance between two notes.

Common intervals:

| Interval | Example |
|---|---|
| Minor 2nd | C → Db |
| Major 2nd | C → D |
| Minor 3rd | C → Eb |
| Major 3rd | C → E |
| Perfect 5th | C → G |
| Octave | C → C |

Harmony is heavily based on intervals.

---

# Consonance vs Dissonance

## Consonance
Stable and pleasant sounding.

Examples:

- Major 3rd
- Perfect 5th

## Dissonance
Tension or unstable sound.

Examples:

- Minor 2nd
- Tritone

Good harmony balances both.

---

# How To Calculate Harmony From a Melody

There is no single exact answer.  
Harmony depends on:

- Scale
- Key
- Chord progression
- Musical style

But there are common methods.

---

# Step 1 — Find the Key

Example melody:

C - E - G - A - F

Possible key:

C Major

Scale notes:

C D E F G A B

---

# Step 2 — Find Chords That Fit the Melody Note

For each melody note, find chords containing that note.

Example:

Melody note: E

Possible chords:

| Chord | Notes |
|---|---|
| C Major | C E G |
| A Minor | A C E |
| E Minor | E G B |

---

# Step 3 — Choose Harmonizing Notes

If melody note is:

C

Possible harmony notes:

- E (major 3rd above)
- G (perfect 5th above)
- A (6th)
- Lower octave C

---

# Basic Harmony Rules

## Rule 1
Harmony notes usually belong to the same scale.

## Rule 2
3rds and 6ths sound smooth.

Example:

Melody: C D E  
Harmony: E F G

---

## Rule 3
Avoid random clashes unless intentionally creating tension.

---

# Parallel Harmony

A simple method:

Take every melody note and add a fixed interval.

Example using 3rds:

| Melody | Harmony |
|---|---|
| C | E |
| D | F |
| E | G |
| F | A |

This creates a pleasant sound.

---

# Chord-Based Harmony

More advanced harmony uses chord progressions.

Example progression:

| Chord | Melody Note |
|---|---|
| C Major | E |
| G Major | D |
| A Minor | C |
| F Major | A |

Harmony notes are chosen from the active chord.

---

# Counterpoint

Counterpoint is advanced harmony where multiple melodies work together independently.

Rules include:

- Avoid parallel fifths
- Resolve dissonance carefully
- Maintain melodic independence

Used heavily in classical music.

---

# Voice Leading

Voice leading means moving harmony notes smoothly.

Bad voice leading:

C → A#

Good voice leading:

C → B

Smaller movements sound more natural.

---

# Example Full Harmony Construction

## Melody

C - D - E - G

## Step 1 — Key

C Major

## Step 2 — Add 3rd Above

| Melody | Harmony |
|---|---|
| C | E |
| D | F |
| E | G |
| G | B |

## Result

Two-note harmony:

- C + E
- D + F
- E + G
- G + B

This creates a smooth harmonized line.

# In Modern Music

## Pop
Uses simple chord harmony.

## Jazz
Uses extended chords:

- 7ths
- 9ths
- 11ths
- 13ths

## Classical
Uses strict voice leading and counterpoint.

## Metal/Rock
Often uses power chords:

Root + 5th

Example:

C + G

---

# Practical Workflow for Harmonizing a Melody

1. Find the key
2. Identify melody notes
3. Determine possible chords
4. Choose harmony intervals
5. Check for clashes
6. Smooth the movement between notes
7. Adjust based on emotion/style

---

# Simple Formula Cheat Sheet

## Major Chord

Root + 4 semitones + 7 semitones

Example:

C + E + G

---

## Minor Chord

Root + 3 semitones + 7 semitones

Example:

A + C + E

---

## Harmony in 3rds

Melody note + note 2 scale positions higher

Example:

C → E  
D → F  
E → G

---

# Final Concept

Melody is usually the main story.

Harmony gives:

- depth
- emotion
- tension
- atmosphere
- richness

A melody alone can sound empty.  
Harmony gives it dimension.
